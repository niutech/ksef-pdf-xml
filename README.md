# Pobieracz e-Faktur KSeF z PDF do XML

Pobierz plik XML e-faktury z KSeF na podstawie wizualizacji w pliku PDF.

Narzędzie działa po stronie klienta, nie przechowuje danych na żadnych serwerach, łączy się z oficjalnym serwerem KSeF za pomocą CORS proxy.

#### [Przetestuj online](https://ai-gov.pl/ksef-xml/)
