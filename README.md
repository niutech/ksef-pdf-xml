# Pobieracz e-Faktur KSeF z PDF do XML

Pobierz plik XML e-faktury z KSeF na podstawie wizualizacji w pliku PDF.

Narzędzie działa po stronie klienta, nie przechowuje żadnych danych na serwerach, łączy się z oficjalnym serwerem KSeF za pomocą CORS proxy, wykorzystuje bibliotekę [pdf-parse](https://github.com/mehmet-kozan/pdf-parse).

#### [Przetestuj online](https://ai-gov.pl/ksef-xml/)
